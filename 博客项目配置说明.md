## 路由配置

### 前端路由

| 路径 | 页面 | 说明 |
|------|------|------|
| /blog/login | Login.vue | 登录页 |
| /blog/register | Register.vue | 注册页 |
| /blog/home | BlogHome.vue | 首页（文章列表） |
| /blog/article/:id | ArticleDetail.vue | 文章详情页 |
| /blog/editor | ArticleEditor.vue | 写文章 |
| /blog/editor/:id | ArticleEditor.vue | 编辑文章 |
| /blog/user/:id | UserCenter.vue | 个人中心 |

### 后端API路由

#### 用户模块 (`/psi/blog/user`)

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /register | 用户注册 |
| POST | /login | 用户登录 |
| GET | /info/{id} | 获取用户信息 |
| PUT | /update | 更新用户信息 |
| PUT | /changePassword | 修改密码 |

#### 文章模块 (`/psi/blog/article`)

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /list | 获取文章列表 |
| GET | /{id} | 获取文章详情 |
| POST | /add | 创建文章 |
| PUT | /edit | 更新文章 |
| DELETE | /{id} | 删除文章 |
| POST | /like/{id} | 点赞文章 |
| DELETE | /like/{id} | 取消点赞 |
| POST | /collect/{id} | 收藏文章 |
| DELETE | /collect/{id} | 取消收藏 |

#### 评论模块 (`/psi/blog/comment`)

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /list/{articleId} | 获取评论列表 |
| POST | /add | 添加评论 |
| DELETE | /{id} | 删除评论 |
| POST | /like/{id} | 点赞评论 |

## 开发说明

### 前后端分离架构

- 前端请求后端API通过 `vue-blog/src/utils/request.js` 配置
- `baseURL` 配置为 `http://localhost:8080/jeecg-boot`
- 请求头携带 `X-User-Id` 和 `X-Username` 用于用户认证

### 数据库表结构

- `sys_blog_user` - 用户表
- `psi_blog_article` - 文章表
- `psi_blog_comment` - 评论表
- `psi_blog_like` - 点赞表
- `psi_blog_collect` - 收藏表
- `psi_blog_category` - 分类表

## 常见问题

### 1. 数据库连接失败

检查 `application-dev.yml` 中的数据库连接配置

### 2. 前端请求404

确认后端已启动，且 `request.js` 中的 `baseURL` 配置正确

### 3. 登录失败

检查 `sys_blog_user` 表中的测试账号数据是否正确
